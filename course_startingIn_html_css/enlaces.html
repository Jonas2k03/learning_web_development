Listas en HTML

Existen (hasta un detemrinado momento en el curso), 2 tipos de listas:

Las listas ordenadas

  1. Siguen un orden
  2. Contienen enumeración
  3. Sin embargo, todo luego se podrá editar con CSS
  4. Su etiqueta es ol

Las listas no ordenadas

  • No siguen un orden
  • Se usan viñetas
  • También se podrá editar con CSS para darle múltiples diseño y funcionalidades
  • Su etiqueta es ul

Para añadir elementos a cualquier tipo de las dos listas presentadas, Se usa el elemento (también etiqueta) li
