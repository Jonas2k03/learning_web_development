Principales componentes de una página HTML

Head

El Head es la cabecera de la página. Se compone principalmente de propiedades no visibles o metadatos, como por ejemplo, el título

Body

El body es el segundo componente de una página HTML. Contiene todos los elementos visibles que se presentarán en la interfaz. En él, se definen una gran cantidad de elementos que mejoran la calidad de la página, cada uno con sus respectivos propósitos. A continuación, se presenta variedad de estos elementos
